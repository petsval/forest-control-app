METSATOO KONTROLL - TOOVERSIOON

1. Paki ZIP lahti.
2. Ava kaust forest_control_app_v13_tooversioon.
3. Tee topeltklikk run_windows.bat.
4. Must aken peab jaama lahti.
5. Kontori arvutis ava: http://127.0.0.1:8010
6. Mine menuusse Lingid: http://127.0.0.1:8010/admin/lingid
7. Kopeeri igale vedukamehele tema link.

Telefoniga kasutamine samas WiFi/vorgus:
- 127.0.0.1 tootab ainult selles arvutis, kus programm kaib.
- Vedukamehe telefonis peab link olema arvuti IP-ga, nt http://192.168.1.25:8010/vedukamees?driver_id=2
- Arvuti IP leiad Windowsis: Start -> kirjuta cmd -> ipconfig -> IPv4 Address.

Kui vedukamehed on metsas mobiilse internetiga:
- see lokaalne variant ei ole piisav;
- programm tuleb panna pilveserverisse voi kontori serverisse avaliku aadressiga;
- siis saab anda neile pysiva lingi.
